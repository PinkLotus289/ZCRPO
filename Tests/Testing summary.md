# Отчет о тестировании — MovieMate

## 1. Общая информация

Данный документ содержит результаты автоматизированного тестирования backend-части приложения **MovieMate**.  
Все основные модули приложения проверены с использованием **pytest** и **unittest.mock**.

## 2. Компоненты системы

| Компонент | Технология |
|-----------|-----------|
| Backend | Flask 3.0 (Python 3.11+) |
| Хранилище данных | JSON файлы (JsonStorage) |
| API интеграция | TMDb API |
| Frontend | HTML5, CSS3, JavaScript (Vanilla JS) |
| Тестирование | pytest, unittest.mock |
| HTTP клиент | Flask test_client |

### Рекомендации по улучшению
1. Добавить интеграционные тесты с реальным TMDb API (использование fixtures)
2. Реализовать end-to-end тесты для проверки работы frontend с backend
3. Добавить тесты производительности для больших коллекций (>1000 фильмов)
4. Внедрить pytest-cov для автоматического отчета о покрытии кода
5. Настроить CI/CD pipeline на GitHub Actions для автоматического запуска тестов
6. Добавить валидацию входных данных API (формат email, длина полей)
