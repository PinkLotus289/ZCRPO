# Результаты тестирования — MovieMate

## 1. Общая информация

Автоматизированное тестирование backend было выполнено с использованием `pytest` и `unittest.mock` для всех API эндпоинтов приложения. Набор тестов охватывает управление пользователями, работу с коллекциями фильмов, поиск и рекомендации.

**Тестовое окружение**
- Backend: Flask 3.0 (Python 3.11+)
- Тестовый фреймворк: pytest
- Мокирование: unittest.mock
- HTTP тестирование: Flask test_client
- Хранилище: JSON файлы
- API: TMDb API
- Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
